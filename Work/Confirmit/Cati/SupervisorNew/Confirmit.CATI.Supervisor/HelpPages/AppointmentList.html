Appointment Activity View Dialog Help

The Appointment List window provides an overview of appointments across open surveys. The list on the left contains total counts for the time periods specified; the list on the right contains specific details about the appointments themselves. All columns are sortable by clicking on the column headers. The list can be filtered by selected surveys, interval periods can be set and alerts can be defined which may result in highlighting of certain appointments. When “Respondent timezone” is checked, the appointment time in the respondent’s local timezone is displayed; otherwise the site local timezone is used.

The columns contain the following information:

  • Interview ID – Respondent ID.
  • Survey ID
  • Survey Name
  • Interviewer name
  • Appointment time – Displayed in either respondent or site local timezone.
  • Timezone – The timezone the respondent is associated with.

A yellow warning triangle is displayed if the appointment is about to be due, and a red x if it is overdue.

An action button exists to activate a call directly from the appointment list. When a survey is closed its entry is removed from the Appointment List. The Appointment List automatically refreshes every minute and it can be refreshed on demand by clicking on the “Refresh” button. Clicking “Export” will export the current Appointment List to Excel file-format.
