Interviewer Activity View Dialog Help

The Interviewer List window contains the list of interviewers currently logged into the system. All columns are sortable by clicking on the column headers. The list can be filtered by selected surveys, and alerts can be defined which may result in highlighting of certain interviewers. When “Alerts on top” is checked, any alerted interviewers are always placed at the top of the list irrespective of the column sort order.

The columns contain the following information:

  • ID – Respondent ID.
  • Survey ID – ID of the survey the interviewer is working on.
  • Survey Name – Name of the survey the interviewer is working on.
  • Interviewer – Interviewer login name.
  • Question – Name of the currently active question.
  • Last submission – Time used to submit the last answer (in seconds).
  • Duration – Duration time of the current state (in HH-MM-SS format).
  • Delivered – Timestamp indicating when the current interview was delivered to the interviewer, or when the break started (format determined by the system locale).
  • Timezone – Respondent time zone.
  • Login – Current interviewer's status ("Yes" indicates the interviewer is logged in and working, "On break" indicates the interviewer is logged in but has left the desk and will be back shortly).
  • State – The current state of the CATI Interviewer Console (the interviewer may be selecting a survey/respondent, conducting an interview, waiting for a call to be assigned, etc…).
  • Keep Alive – Timestamp of the last successful server-to-interviewer-station query, indicates that the current interviewer session is considered as still continuing (when time interval between the last successful keep-alive query and the last unsuccessful query exceeds the limit, the interviewer is automatically logged out).
  • Dialer – Indicates whether the interviewer is logged into the dialer or not.
  • Dialing mode – The dialing mode which the interviewer is currently working in.
  • Problem – Displays a corresponding error message in case a telephony problem occurs.
  • Station ID – The station ID for the console the interviewer is logged into.

Action buttons exist to terminate the interviewer’s session, send a message to the interviewer or launch live monitoring. When an interviewer logs out of the system, or when they are logged out by the system automatically for any reason (this is also the case when an interviewer's job is terminated by the supervisor), their entry is removed from the Interviewer List. The refresh interval for the list can be defined from the dropdown, and it can be refreshed on demand by clicking on the “Refresh” button. Clicking “Export” will export the current Interviewer List to Excel file-format.
