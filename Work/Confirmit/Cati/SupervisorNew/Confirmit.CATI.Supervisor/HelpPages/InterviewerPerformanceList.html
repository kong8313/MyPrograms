Performance Activity View Dialog Help

The Performance List window contains performance data regarding interviewers that have worked today. All columns are sortable by clicking on the column headers. When “Logged in interviewers only” is checked, only interviewers who are currently logged into the system are shown. The checkbox "Breakdown by survey" allows to switch between modes with and without breakdown by surveys. When “Active surveys only” is checked, only surveys within which interviewers are working are displayed.

The columns contain the following information:

  • Interviewer
  • Survey ID
  • Survey name
  • Interviewing time – The total time (excluding breaks) that has been spent interviewing today (HH:MM:SS).
  • Interviews – The number of interviews performed today.
  • Interviewer name
  • Completes – The number of completed interviews performed today.
  • Strike rate(last hour) – The current strike rate (completes per hour) based on the last hour.
  • Strike rate(average) – The average number of completes per hour for time spent working on the survey today.

The performance data is aggregated every 3 minutes and the view is updated every minute. Click Refresh to force an update of the view. Click “Export” to export the current Performance List to Excel file-format.
