Surveys Activity View Dialog Help

The Survey List window contains the list of all open CATI surveys. All columns are sortable by clicking on the column headers. The list can be filtered by selected surveys, and alerts can be defined which may result in highlighting of certain surveys. When “Active surveys only” is checked, only surveys within which interviewers are working are displayed.

The columns contain the following information:

  • Survey ID
  • Name
  • Logged in – The number interviewers currently logged in.
  • Assigned – The number of assigned interviewers.
  • Target – The target for a total number of completed interviews.
  • Time today – The total time that has been spent interviewing today (HH:MM:SS).
  • Total time – The total time that has been spent interviewing (HH:MM:SS).
  • Next appointment – Time of the first due appointment.
  • Active – The number of active calls.
  • Scheduled – The number of scheduled calls.
  • Not scheduled – The number of not-scheduled interviews.
  • Strike rate – The current strike rate (completes per hour) based on the last 15 minutes activity.
  • Strike rate (1h) – The current strike rate (completes per hour) based on the last 1 hour activity.
  • Calls per hour – The call rate (calls per hour) based on the last 15 minutes activity.
  • Calls per hour (1h) – The call rate (calls per hour) based on the last 1 hour activity.
  • Duration – The average duration of calls based on the last 15 minutes activity.
  • Duration (1h) – The average duration of calls based on the last 1 hour activity.

Action buttons exist to view a breakdown of calls by extended status and to send a message to all interviewers working on the survey. When a survey is closed its entry is removed from the Survey List. The refresh interval for the list can be defined from the dropdown, and it can be refreshed on demand by clicking on the “Refresh” button. Clicking “Export” will export the current Survey List to Excel file-format.
