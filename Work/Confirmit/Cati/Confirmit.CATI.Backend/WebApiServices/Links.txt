Short instruction to use Postman
1. Open in a browser the following url: http://localhost/Confirmit/WebServices/18.0/LogOn.asmx
2. Press on LogOnUser link
3. Type login and password and press Enter.
4. You will see the key like "5e90adba-a046-4a46-8c2d-7d1844282f47&7CA62A38E459D9E982BA82FF2A...FA23104EA509E+1". Keep it.
5. Open Postman and specify url to rest api like the following: http://localhost/catiapi/companies/1/callhistory
6. Specify header with key="X-Confirmit-ApiKey" and value=<the received key>) and press send
7. You can try to use otada filter like the following: 
http://localhost/catiapi/companies/1/callhistory?$filter=SurveyId eq 'p100001' 
http://localhost/catiapi/companies/1/callhistory?$filter=CallCenterId eq 1 and InterviewId eq 11
