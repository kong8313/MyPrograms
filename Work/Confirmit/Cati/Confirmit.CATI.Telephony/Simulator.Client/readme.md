# CATI Simulator Control Panel
The project is new Web extension for our CATI Simulator which should be available by <http://localhost:3838/catidialersimulator/app> url

### Manual update
To update source code manually, you have to run `npm run build` and copy `/build` folder content into `/App_Data/www` folder of simulator webservice

### Run
To start project run `npm run start` command
